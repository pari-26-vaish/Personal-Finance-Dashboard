## 📖 Overview

**Personal Finance Dashboard** is a lightweight, privacy-first web app for tracking income and expenses. There's no signup, no server, and no data collection — everything runs and stays entirely on your device. It's built for anyone who wants a fast, no-friction way to see exactly where their money goes each month.

---

## ✨ Key Features

| Feature | Description |
|---|---|
| 🔒 **100% Private** | All data is stored locally on your device — nothing is ever sent to a server |
| 📊 **Spending Visualization** | Interactive charts break down spending by category (Food, Bills, Transport, and more) |
| 🗂️ **9+ Categories** | Organize transactions into meaningful, customizable categories |
| ⚡ **No Signup Required** | Open it and start tracking — no accounts, no friction |
| 📱 **Clean, Fast UI** | Minimal interface focused on clarity, not clutter |
| 🌗 **Light/Dark Mode** | Toggle between themes for comfortable viewing anytime |

## 🚀 Getting Started

### Prerequisites
Just a modern web browser — no installations, dependencies, or build tools required.

### Installation

```bash
# Clone the repository
git clone https://github.com/pari-26-vaish/Personal-Finance-Dashboard.git

# Navigate into the project folder
cd Personal-Finance-Dashboard

# Open in your browser
open index.html   # macOS
start index.html   # Windows
```

Or simply try it live: **[Launch Dashboard →](https://pari-26-vaish.github.io/Personal-Finance-Dashboard/)**
